====================================================================================================
四、文件選擇指南
====================================================================================================

如果你想要:

1. 了解GNN增強的整體效果（與原始模型和參考模型比較）
   → 使用: gnn_only_ranking_*.txt (原始排名)

2. 選擇最佳的GNN配置（只關注GNN變體之間的比較）
   → 使用: gnn_relative_ranking_*.txt (相對排名，按分類)
   → 或: gnn_overall_ranking_*.txt (相對排名，整體)

3. 快速了解整體表現和選型建議
   → 使用: GNN整體排名綜合分析.txt

4. 了解特定數據集分類下的最佳配置
   → 使用: gnn_relative_ranking_*.txt (按10種分類詳細排名)

5. 了解跨所有數據集的平均表現
   → 使用: gnn_overall_ranking_*.txt (不區分分類)

====================================================================================================
五、關鍵差異總結
====================================================================================================

按分類排名 vs 整體排名:

按分類排名 (gnn_relative_ranking_*.txt):
✓ 優點: 能看到在特定類型數據集上的表現
✓ 優點: 幫助選擇針對特定場景的最佳配置
✓ 缺點: 需要查看10個分類的排名
✓ 適用: 當你知道數據集特徵時

整體排名 (gnn_overall_ranking_*.txt):
✓ 優點: 一個排名覆蓋所有數據集
✓ 優點: 快速了解整體最佳配置
✓ 優點: 包含統計分析（按模型、按階段）
✓ 缺點: 可能掩蓋在特定類型數據上的優勢
✓ 適用: 當你需要通用的最佳配置時

====================================================================================================
六、數據統計
====================================================================================================

數據集:
- 總數: 116個
- 分類: 10種
  * 按大小: large_datasets (20個), small_datasets (96個)
  * 按任務: binclass, multiclass, regression
  * 按特徵: numerical, categorical, balanced

模型:
- GNN可插入模型: 9個
  (excelformer, fttransformer, resnet, tabnet, tabtransformer, 
   trompt, vime, scarf, subtab)
- GNN階段: 5種
  (start, materialize, encoding, columnwise, decoding)
- GNN變體總數: 45個 (9 × 5)

競爭者:
- 原始排名: 118個 (包含原始模型和參考模型)
- 相對排名: 45個 (只包含GNN變體)

====================================================================================================
七、推薦閱讀順序
====================================================================================================

初次使用者:
1. 先閱讀: GNN整體排名綜合分析.txt
2. 再查看: gnn_overall_ranking_0.05-0.15-0.8.txt (或你的訓練比例)
3. 如需詳細: gnn_relative_ranking_*.txt (查看特定分類)

進階使用者:
1. 直接查看: gnn_overall_ranking_*.txt
2. 對照查看: gnn_relative_ranking_*.txt (特定分類)
3. 深入分析: gnn_only_ranking_*.txt (與基準模型比較)
